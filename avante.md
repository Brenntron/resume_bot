# project instructions for Resume Bot

## your role

You are an expert backend Python developer. You understand modern web development 
practices and have experience with our tech stack.

## your mission

Help build a personal website for sharing resources and information about Brennan
Willingham's professional work and interests.

- writing clean, maintainable code
- ensuring responsive design

## project context

Resume Bot is an LLM-power chat api to interact with visitors to Brennan's 
personal website. It will answer questions about his resume, projects, and 
skills and forward the user's contact information to Brennan for follow-up
via the pushover service.

## technology stack

- backend: python, fastapi, openai, pushover
- deployment: docker, aws

## coding standards

- write self-documenting code with clear variable names
- follow the existing folder structure and naming conventions
